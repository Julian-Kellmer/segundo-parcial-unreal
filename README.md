# Segundo Parcial — Unreal Engine

Proyecto desarrollado para el segundo parcial de la materia Unreal Engine.

## Estructura del proyecto

```
Content/          → Assets, materiales, blueprints, mapas
Config/           → Configuraciones del proyecto
Source/           → Código fuente (C++ si aplica)
```

## Requisitos

- Unreal Engine 5.x
- Windows 10/11

## Cómo abrir el proyecto

1. Clonar el repositorio
2. Hacer doble click en el archivo `.uproject`
3. Unreal Engine abre y compila automáticamente
